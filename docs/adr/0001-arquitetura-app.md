# ADR 001 — Escolha da arquitetura do app

## Decisão
Utilizar arquitetura MVC para o aplicativo mobile.

## Contexto
Equipe tem prazo curto e familiaridade com MVC; complexidade baixa neste MVP.

## Alternativas consideradas
- MVVM
- Clean Architecture

## Consequências
+ Implementação simples e rápida
- Menos escalável para futuras camadas
